# 梁強度計算WEB

Excel v9「コの字2追加版」をブラウザ用に移植した、梁強度計算用の静的WEBページです。

## 使い方

1. `index.html` をブラウザで開きます。
2. 形鋼プリセット、材料、支持・荷重ケース、断面方向、荷重、寸法を入力します。
3. 計算結果、断面イメージ、計算式メモ、簡易モーメント図が自動更新されます。
4. 「印刷 / PDF」で計算結果をPDF保存できます。
5. 「CSV保存」で計算結果をCSV保存できます。
6. 「入力JSON保存」で現在の入力値を保存できます。

## ファイル構成

- `index.html` : 画面本体
- `style.css` : デザイン・レスポンシブレイアウト
- `data.js` : Excelから移植した材料DB、ケースDB、形鋼DB、初期値
- `app.js` : 断面計算、梁強度計算、描画、保存処理

## 主な機能

- 形鋼プリセット選択
- 手入力断面計算
- L字、コの字、コの字2、I形鋼、H形鋼、長方形、丸棒に対応
- X方向 / Y方向切替
- 自重等分布荷重の加算
- 最大曲げモーメント計算
- 最大曲げ応力計算
- 安全率判定
- 外力たわみ・自重たわみ・合計たわみ計算
- 許容たわみ判定
- 断面イメージ描画
- 簡易モーメント図描画
- 左右表示 / 上下表示切替

## 単位

- 長さ：mm
- 荷重：N
- 等分布荷重：N/mm
- 応力：N/mm²
- 断面積：mm²
- 断面二次モーメント：mm⁴
- 断面係数：mm³
- 密度：kg/m³

## 注意

本ページはアップロードされたExcelの簡易計算ロジックをブラウザ用に移植したものです。
穴あき、溶接、局部座屈、応力集中、疲労、せん断変形、動荷重などは未考慮です。
正式運用前に、既存Excel・社内基準・実機条件で照合してください。

## GitHub Pagesで公開する場合

GitHubのリポジトリ画面で以下を設定してください。

1. `Settings`
2. `Pages`
3. `Build and deployment`
4. `Source`: `Deploy from a branch`
5. `Branch`: `main`
6. `Folder`: `/ root`
7. `Save`

数分後、以下のURLで開けます。

```text
https://sa-ban777.github.io/beam-strength/
```
